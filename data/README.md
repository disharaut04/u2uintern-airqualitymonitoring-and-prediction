# Data Folder Documentation
This project uses a single dataset for Air Quality Monitoring and Prediction.

## Dataset
'aqi_dataset.csv' -> Main dataset containing air quality parameters such as pollutant levels, location, and timestamps.

## Usage 
 This dataset is used for: 
-Data preprocessing and cleaning
-Exploratory Data Analysis (EDA)
-Model training and evaluation
-Storing structured data for future database integration

## Note
Currently, only CSV format data is used in this project. No JSON or PDF datasets are included.
